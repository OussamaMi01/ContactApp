# ContactApp

**ContactApp** is a simple console-based application for managing a list of contacts. Users can add, view, edit, and delete contacts through an intuitive command-line interface.

## Features

- **View Contacts**: Displays a list of all saved contacts with details.
- **Add Contact**: Allows users to add new contacts with optional fields.
- **Edit Contact**: Update existing contact information.
- **Delete Contact**: Remove a contact from the list.
- **User-Friendly Interface**: Clear instructions and feedback for each operation.

## Requirements

- **C#**: The application is written in C# and requires a compatible runtime environment to execute.

## How to Run

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/OussamaMi01/ContactApp.git
   cd ContactApp
   ```

2. **Compile and Run**:
   Use the C# compiler to build and run the application:
   ```bash
   dotnet run
   ```

3. **Follow the On-Screen Instructions**:
   - Use the menu to navigate and perform operations.

## File Structure

- `Program.cs`: Main file containing the application logic.


## Example Usage

1. Add a new contact:
   - Enter the first name, last name, phone number, and email. Any of these fields can be left blank.
   
2. View contacts:
   - Displays a list of all stored contacts.
   
3. Edit a contact:
   - Choose a contact by index and update any of its details.
   
4. Delete a contact:
   - Select a contact by index to remove it.
